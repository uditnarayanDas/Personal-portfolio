# Personal Portfolio

Welcome to my personal portfolio! This project showcases my work, skills, and creative journey as a developer.

## 🚀 Overview

Built with modern technologies, this portfolio is designed to be highly interactive, visually stunning, and performance-oriented. It features smooth animations, responsive design, and a minimalist aesthetic.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & [Tabler Icons](https://tabler-icons.io/)
- **Components**: Custom UI components with a "Liquid Glass" aesthetic

## ✨ Key Features

- **Dynamic Hero Section**: Interactive background and signature typography.
- **Bento Grid**: A modern way to showcase various aspects of my professional life.
- **Project Showcase**: Detailed view of my recent works with sticky scroll effects.
- **Interactive Connect**: Unique "Let's Connect" button and easy email copy functionality.

## 🏃 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
